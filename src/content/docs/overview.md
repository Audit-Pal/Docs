# AuditPal Overview

AuditPal is a continuous security platform for smart contracts. It monitors deployed contracts in real time, equips human auditors with an AI co-pilot for deeper and faster analysis, and runs a competitive agent network that improves automatically as the threat landscape evolves.
Most security stops at deployment. AuditPal does not.
The platform is built around two core products: Continuous Security Agents that watch deployed contracts around the clock, and an Auditor Co-Pilot that handles the heavy analytical lifting so human auditors can focus on the reasoning that actually requires a human. Both are powered by a Bittensor subnet where competing agents are scored on real, reproducible findings, not heuristics or reputation.
